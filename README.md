# 拾纷电商管理系统（微信小程序）

这是一个天猫风格的仓库管理微信小程序原型，面向电商仓库日常的库存、入库、退货入库、出库和成本登记。

## 主要功能

- 公司代码登录：进入系统前需要输入公司代码，默认公司代码为 `sfds`，登录页不会展示默认代码。
- 首页仪表盘：展示代发总成本、库存数量、库存总金额、库存预警、供应商货款余额、今日出入库、本月出入库数量与金额。
- 库存管理：登记商品名称、规格、单价、在库数量、供应商、备注和预警数量，支持编辑和一键全选。
- 供应商货款余额：支持普通渠道商、3W品牌渠道商、台州伊文渠道商下拉选择并登记余额。
- 入库管理：支持日期筛选，从库存商品下拉选择后登记入库信息，支持编辑和一键全选。
- 退货入库管理：支持日期筛选和库存商品下拉选择，登记订单、物流和货物状态，支持编辑和一键全选。
- 出库管理：支持日期筛选和库存商品下拉选择，登记出库数量、单价、总金额和用途/备注，支持编辑和一键全选。
- 成本登记：支持代发成本、采购成本、人工成本、其他成本分类下拉选择，支持编辑和一键全选。

## 运行方式

1. 使用微信开发者工具导入当前目录。
2. 选择小程序项目并使用测试 AppID 或自己的 AppID。
3. 编译后在登录页输入公司代码进入系统。

## 数据存储

当前版本使用微信小程序本地缓存 `wx.setStorageSync` / `wx.getStorageSync` 保存数据，适合原型演示和单机使用。
